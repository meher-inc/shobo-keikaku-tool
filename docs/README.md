## このディレクトリの目的

本ディレクトリは plan.todokede.jp の開発・運用に関わる **戦略・技術・法務ドキュメント**を集約します。原本は戦略プロジェクト（Claude triv Strategy Project）側にあり、本ディレクトリはその **開発側ミラー**です。

実装中に「料金いくらだっけ」「LPのFAQどうなってたっけ」と参照する際の、開発プロジェクト内の一次参照先として使います。

## 運用ルール

1. **原本は戦略プロジェクト側**：本ディレクトリのファイルを直接編集しない
2. **同期タイミング**：戦略側で v{n+1} が確定したら、開発側にも配置してcommit
3. **リネーム規則**：戦略側の番号プレフィックス（`06_` など）は落とし、英語スラッグ + バージョンサフィックスに統一
4. **破棄ルール**：旧バージョンは `archive/` に移動、削除はしない
